# Advanced Modular Negotiation Agent System Prompt

I'm looking to build an advanced modular negotiation-focused AI agent system using Concordia. I want to create sophisticated negotiation agents that can be composed from swappable, specialized modules to handle different negotiation scenarios and research applications.

The system should have a base negotiation agent prefab that works standalone with core negotiation capabilities, plus a collection of advanced negotiation modules that can be added, removed, or mixed and matched to create uniquely powerful negotiation agents.

Before we start building this out though, I want to do a little planning with you. I first want you to make a new file called projectplan.md. Please build an in-depth plan for the app with high level checkpoints for each major step and features, then in each checkpoint have a broken down list of small tasks you'll need to do to complete that checkpoint. Also in this plan include instructions we will give to each agent that will need to accomplish the following:

**Cross-Cultural and Multi-Language Negotiation**
- Cultural adaptation component that can switch between collectivist/individualist styles
- High-context vs low-context communication patterns
- Adaptation to different cultural negotiation norms and expectations

**Temporal Dynamics and Long-term Strategy**
- Multi-horizon planning that considers short-term tactics, medium-term relationship building, and long-term reputation effects
- Dynamic strategy adjustment based on negotiation phase and relationship history
- Understanding of how current actions affect future negotiation power

**Multi-Agent Collective Intelligence**
- Swarm negotiation system where specialized sub-agents collaborate
- Different agents for market analysis, emotional intelligence, game theory strategy, and diplomatic relationship management
- Democratic decision making and consensus building between the specialized agents

**Probabilistic Reasoning Under Uncertainty**
- Bayesian belief updating about opponent preferences and constraints
- Quantified uncertainty in power assessments and market conditions
- Planning under multiple possible scenarios with incomplete information

**Dynamic Strategy Evolution with Reinforcement Learning**
- Meta-learning for few-shot adaptation to new negotiation contexts
- Continual learning that improves without forgetting past experiences
- Automatic discovery of novel negotiation tactics through experience

**Multi-Modal Negotiation with Theory of Mind**
- Recursive theory of mind reasoning ("I think that you think that I think...")
- Emotional intelligence to detect frustration, eagerness, deception in counterparts
- Dynamic belief state tracking of what others believe about the situation

I'd like this to be built using the Concordia framework, leveraging its prefab system, component-based architecture, and associative memory capabilities. The system should be modular so researchers can easily compose different combinations of negotiation capabilities for various experimental scenarios.

Before we start building this out though, I want to do a little planning with you. I first want you to make a new file called projectplan.md that analyzes the Concordia codebase, understands how prefabs and components work, and creates a detailed implementation plan for building this modular negotiation agent system with all the advanced components I described.
